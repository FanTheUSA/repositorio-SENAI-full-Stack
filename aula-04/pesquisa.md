1. **O que é uma estrutura de repetição?**
   É uma estrutura que permite executar um mesmo conjunto de instruções várias vezes.

2. **Para que serve o laço for?**
   Serve para repetir um bloco de código quando já sabemos quantas vezes a repetição deve acontecer.

3. **Para que serve o laço while?**
   Serve para repetir um bloco de código enquanto uma condição for verdadeira.

4. **Para que serve o laço do-while?**
   Serve para repetir um bloco de código enquanto uma condição for verdadeira, garantindo que ele seja executado pelo menos uma vez.

5. **Qual é a diferença entre for, while e do-while?**
   O `for` é usado quando a quantidade de repetições é conhecida. O `while` é usado quando a repetição depende de uma condição. O `do-while` funciona como o `while`, mas executa o bloco uma vez antes de verificar a condição.
