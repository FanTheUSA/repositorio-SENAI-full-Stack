O que são operadores em programação?
São símbolos usados para realizar operações com valores e variáveis, como somar, comparar ou verificar condições.
Para que servem os operadores aritméticos? Cite exemplos.
Servem para fazer cálculos matemáticos. Exemplos: + (soma), - (subtração), * (multiplicação), / (divisão) e % (resto da divisão).
Qual a diferença entre / e % em JavaScript?
/ faz a divisão. Já % retorna o resto da divisão.
Como descobrir se um número é par usando o operador %?
Se numero % 2 == 0, o número é par.
O que é uma expressão em programação?
É uma combinação de valores, variáveis e operadores que produz um resultado.
O que significa precedência de operadores?
É a ordem em que os operadores são executados em uma expressão.
Por que os parênteses podem mudar o resultado de uma expressão?
Porque eles fazem a operação dentro deles ser executada primeiro.
Qual a diferença entre =, == e ===?
= atribui um valor.
== compara apenas o valor.
=== compara valor e tipo.
Por que é recomendado usar === em JavaScript?
Porque a comparação é mais precisa, evitando resultados inesperados.
O que os operadores relacionais retornam: texto, número ou boolean?
Retornam valores booleanos: true ou false.
Quando usamos o operador >? E o operador >=?
> quando queremos saber se um valor é maior que outro.
>= quando queremos saber se é maior ou igual.
Quando usamos o operador <? E o operador <=?
< quando queremos saber se um valor é menor que outro.
<= quando queremos saber se é menor ou igual.
Para que serve o operador lógico &&?
Serve para verificar se duas condições são verdadeiras ao mesmo tempo.
Para que serve o operador lógico ||?
Serve para verificar se pelo menos uma das condições é verdadeira.
Para que serve o operador lógico !?
Serve para inverter um valor lógico. Exemplo: true vira false.
Qual a diferença entre uma condição simples e uma condição composta?
A condição simples possui apenas uma verificação. A composta possui duas ou mais condições ligadas por operadores lógicos.
Explique uma situação real em que duas condições precisam ser verdadeiras ao mesmo tempo.
Para entrar em uma prova, o aluno precisa estar inscrito e apresentar um documento de identificação.
Explique uma situação real em que apenas uma entre duas condições precisa ser verdadeira.
Uma pessoa pode entrar em um evento apresentando RG ou CNH.
Como o console.log() pode ajudar a encontrar erros no código?
Ele mostra informações na tela, ajudando a verificar valores de variáveis e entender onde o programa pode estar errando.
Por que devemos testar o mesmo programa com valores diferentes?
Porque um programa pode funcionar com alguns valores e apresentar erros com outros. Testando várias situações, conseguimos garantir que ele funcione corretamente para diferentes casos.
