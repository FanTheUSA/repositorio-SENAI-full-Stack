1. O que é uma estrutura condicional em programação? Dê um exemplo do cotidiano usando "se... então... senão".

É uma forma de o programa tomar decisões.

Exemplo: Se estiver chovendo, então levo guarda-chuva, senão saio sem ele.

2. Qual é a diferença entre usar apenas if e usar if/else?

if: faz algo se a condição for verdadeira.
if/else: faz uma coisa se for verdadeira e outra se for falsa.

3. Quando usamos else if? Crie um exemplo com três ou mais possibilidades.

Usamos quando temos mais de duas opções.

Exemplo:

Se a nota for maior que 9: Excelente.
Senão, se for maior que 7: Bom.
Senão, se for maior que 6: Regular.
Senão: Reprovado.

4. O que significa uma condição retornar true ou false?

true = verdadeiro.
false = falso.

Exemplo: 10 > 5 retorna true.

5. Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?

== compara apenas o valor.
=== compara o valor e o tipo.

O mais recomendado é ===.

6. Para que servem os operadores &&, || e !?

&& = E.
|| = OU.
! = NÃO.

Exemplos:

Maior de 18 e ter carteira.
Passar na prova ou na recuperação.
Não estar bloqueado.

7. O que o operador % faz? Por que ele é útil para descobrir se um número é par ou ímpar?

Ele mostra o resto da divisão.

Se o resto da divisão por 2 for 0, o número é par. Se for diferente de 0, é ímpar.

8. O que é switch case? Em que situação ele fica melhor do que vários else if?

É uma estrutura usada para escolher entre várias opções.

É melhor quando há muitas opções para comparar.

9. Para que servem break e default dentro de um switch?

break: para a execução do caso.
default: executa quando nenhuma opção é encontrada.

10. Por que é importante testar o mesmo programa com vários valores diferentes?

Para ter certeza de que ele funciona corretamente em todas as situações.
