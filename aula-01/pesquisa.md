1. O que é programação?
Programação é o processo de criar instruções para que um computador execute tarefas. Essas instruções são escritas em uma linguagem de programação. Ela é usada para desenvolver sites, aplicativos, jogos e sistemas.

2. O que é um algoritmo?
Um algoritmo é uma sequência de passos para resolver um problema ou realizar uma tarefa. Ele funciona como uma receita de bolo, seguindo uma ordem lógica. Antes de programar, geralmente criamos um algoritmo.

3. Qual a diferença entre algoritmo e linguagem de programação?
O algoritmo é a ideia ou o passo a passo da solução. Já a linguagem de programação é a forma usada para escrever essa solução para o computador entender. Um algoritmo pode ser escrito em várias linguagens.

4. O que é JavaScript?
JavaScript é uma linguagem de programação muito usada no desenvolvimento web. Ela permite criar páginas interativas e dinâmicas. Com ela, é possível adicionar animações, botões e outras funcionalidades.

5. Onde o JavaScript pode ser utilizado atualmente?
O JavaScript pode ser usado em sites, aplicativos para celular e programas para computador. Também é utilizado em servidores por meio do Node.js. Atualmente, está presente em grande parte da internet.

6. O que é o Node.js?
Node.js é uma ferramenta que permite executar JavaScript fora do navegador. Com ele, é possível criar servidores e aplicações completas. Ele é muito usado no desenvolvimento web.

7. Qual é a função do Node.js no desenvolvimento com JavaScript?
O Node.js permite que o JavaScript seja executado no lado do servidor. Isso possibilita criar sistemas completos usando apenas uma linguagem. Ele também ajuda a gerenciar arquivos e conexões de rede.

8. O que é o VS Code?
VS Code é um editor de código criado pela Microsoft. Ele oferece ferramentas que ajudam a programar com mais facilidade. É muito utilizado por estudantes e profissionais.

9. Qual a diferença entre um editor de texto comum e um editor de código?
Um editor de texto comum serve apenas para escrever textos. Já um editor de código possui recursos específicos para programação, como destaque de sintaxe e sugestões de código. Isso facilita o trabalho do programador.

10. O que é o terminal?
O terminal é uma ferramenta que permite executar comandos por texto. Ele é usado para rodar programas, navegar entre pastas e realizar tarefas do sistema. É muito importante para quem programa.

11. Para que serve o comando node arquivo.js?
Esse comando é usado para executar um arquivo JavaScript usando o Node.js. O computador lê e executa as instruções que estão no arquivo. É uma forma simples de testar programas.

12. O que é uma variável?
Uma variável é um espaço usado para armazenar informações em um programa. Ela pode guardar números, textos e outros dados. Seu valor pode ser utilizado e alterado durante a execução.

13. Qual a diferença entre let e const?
let cria uma variável que pode ter seu valor alterado depois. Já const cria uma constante, cujo valor não pode ser modificado. Ambas são formas modernas de declarar variáveis em JavaScript.

14. O que são tipos de dados? Cite exemplos.
Tipos de dados definem o tipo de informação armazenada em uma variável. Alguns exemplos são: número (10), texto ("Olá"), verdadeiro ou falso (true e false). Eles ajudam o programa a tratar os dados corretamente.

15. Qual a principal diferença entre JavaScript e TypeScript?
JavaScript é uma linguagem de programação tradicional e flexível. TypeScript é uma versão do JavaScript que adiciona tipagem de dados. Isso ajuda a encontrar erros mais facilmente durante o desenvolvimento.
