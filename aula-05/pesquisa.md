1. **O que é um array em programação?**
   É uma estrutura que guarda vários valores em uma única variável.

2. **Por que é melhor usar um array do que criar várias variáveis separadas, como aluno1, aluno2 e aluno3?**
   Porque deixa o código mais organizado e facilita trabalhar com vários dados ao mesmo tempo.

3. **O que é índice em um array?**
   É a posição de cada elemento dentro do array.

4. **Por que o primeiro elemento de um array fica na posição 0?**
   Porque as linguagens de programação geralmente começam a contar os índices a partir do zero.

5. **Para que serve o `.length`?**
   Serve para mostrar quantos elementos existem no array.

6. **Como acessamos o primeiro elemento de um array chamado alunos?**
   Usando `alunos[0]`.

7. **Como podemos alterar um valor dentro de um array?**
   Atribuindo um novo valor à posição desejada. Exemplo: `alunos[1] = "Maria"`.

8. **Qual a relação entre arrays e laços de repetição?**
   Os laços ajudam a percorrer todos os elementos do array automaticamente.

9. **Qual a diferença entre percorrer um array com for e apenas mostrar um item específico?**
   Com `for` vemos todos os elementos; mostrando um item específico vemos apenas uma posição.

10. **O que é um acumulador? Dê um exemplo de uso com notas ou valores.**
    É uma variável usada para somar valores. Exemplo: somar todas as notas de uma turma.

11. **Para que servem operadores aritméticos como +, -, * e / dentro de um programa?**
    Servem para fazer cálculos matemáticos.

12. **Para que servem operadores lógicos como && e ||?**
    Servem para combinar condições em testes e decisões.

13. **Qual a diferença entre if/else e switch?**
    O `if/else` permite testar várias condições diferentes; o `switch` é usado para comparar uma variável com vários valores possíveis.

14. **O que os métodos push e pop fazem em um array?**
    `push` adiciona um elemento no final do array e `pop` remove o último elemento.

15. **O que é uma matriz e como ela se relaciona com arrays?**
    Uma matriz é um array que contém outros arrays, formando linhas e colunas de dados.
